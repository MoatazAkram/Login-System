# Login System
A responsive Login System built with HTML, CSS, JavaScript, and Bootstrap. Allows users to Sign Up, Sign In, validates inputs, stores user data in localStorage, and displays a personalized welcome message.

# Features
- Sign Up with validation
- Sign In with validation
- Store user data using localStorage
- Welcome message with user's name after login
- Responsive design using Bootstrap

# Technologies Used
- HTML
- CSS
- Bootstrap 5
- JavaScript
